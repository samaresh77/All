A modern, browser-based Snake game built with a JavaScript Canvas frontend, Python FastAPI backend, MongoDB database, and user authentication system.

## Features

### Game Features
- Classic Snake gameplay with modern visuals
- Multiple difficulty levels (Easy, Medium, Hard, Insane)
- Responsive design that works on desktop and mobile
- Touch controls for mobile devices
- Local high score tracking

### User Features
- User registration and authentication
- Secure password hashing with bcrypt
- JWT token-based authentication
- User statistics tracking (games played, total score, average score)
- Personal best scores

### Leaderboard System
- Global leaderboard showing top 20 scores
- Personal score history
- Real-time score submission
- Filter between global and personal scores

### Technical Features
- FastAPI backend with async/await support
- MongoDB for data persistence
- CORS middleware for cross-origin requests
- Production-ready error handling
- RESTful API design
